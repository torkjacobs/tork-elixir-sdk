    TorkGovernance.Plug (tork_governance v0.1.0)

    Copy Markdown View Source

    Plug middleware for Tork governance.

    Automatically governs incoming request bodies for PII and attaches the governance receipt to conn.assigns[:tork_receipt].

    Usage in a Phoenix Router

    pipeline :governed do
  plug TorkGovernance.Plug
end

scope "/api", MyAppWeb do
  pipe_through [:api, :governed]
  post "/chat", ChatController, :create
end
